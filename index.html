Sidewinder Timeseries Database

Sidewinder is a fully open source Time Series Database available under Apache 2.0 License. It's designed from the ground up with performance, efficiency and scalability in mind.

Our goal is to reduce the cost of storing and analyzing time series data at large scale so valuable business insights can be generated in real-time.

Checkout Sidewinder on Github

Features

Persistent

Memory mapped writes to disk

Fast

Upto ~2 million writes per second per node!

Visualizations

Official Grafana Data Source for visual analytics

Collectors
